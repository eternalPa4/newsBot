# newsBot

Бот для поиска сообщений о происшествиях в городских и региональных Telegram-чатах.

## Настройка через `.env`

1. Скопируйте шаблон:
   ```bash
   cp .env.example .env
   ```
2. Заполните значения `API_ID`, `API_HASH`, `BOT_TOKEN`.
3. При необходимости измените `CITY`, списки чатов, алиасы города и ключевые слова.

## Что улучшено

- Все основные параметры и списки чатов читаются из `.env`.
- Поиск по событиям стал точнее:
  - добавлены синонимы для событий (например, `пожар` → `горит`, `возгорание`);
  - пользовательские сообщения теперь распознаются как запрос по конкретному событию;
  - запросы вида «сегодня» автоматически запускают сводку за 24 часа.
